\title{Hyperbolic discounting note}


\maketitle


The aim is to produce a recursive expression that allows us to apply ADP methods
to hyperbolic discounting.  At this stage we just seek a recursive expression
for lifetime value.

Lifetime value is given by 
%
\begin{equation*}
    v(x) \coloneq 
    \EE_x 
    \left[
        r(X_0) + \alpha r(X_1) + \sum_{t \geq 2} \beta^t r(X_t)
    \right]
\end{equation*}
%
Pointwise this is
%
\begin{equation*}
    v = r + \alpha P r + \sum_{t \geq 2} (\beta P)^t r
\end{equation*}
%
We generalize to allow state-dependent discounting, taking $J$ and $K$ to be
positive linear operators and writing
%
\begin{equation}\label{eq:vdef}
    v = r + J r + \sum_{t \geq 2} K^t r
\end{equation}
%
Rearranging gives
%
\begin{equation*}
    v 
    = r + J r + K^2 r + K \sum_{t \geq 2} K^t r
\end{equation*}
%
Using \eqref{eq:vdef} again gives
%
\begin{align*}
    v & = r + J r + K^2 r + K (v - r - Jr)
    \\
      & = r - Kr + J r - KJr + K^2 r + K v,
\end{align*}
%
or
%
\begin{equation*}
    v = (I - K)(r + J r)  + K^2 r + K v 
\end{equation*}
%
If $r(K) < 1$, then lifetime value is uniquely defined by this recursive
expression.
